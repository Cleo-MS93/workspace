## Variáveis, Tipos de Dados e Operadores Matemáticos em Java

### Oque é uma Variável?
"Um espaço na memória do computador, onde se pode guardar valores"

Existem 4 tipos:
 - instancia: objeto
 - Classe: classe
 - Local: dentro de métodos
 - Parâmetro: na assinatura do método
